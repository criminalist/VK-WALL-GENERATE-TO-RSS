# vk2rss
Generate RSS Feed from VK Group wall [early beta]

## Docker ENV Params
 - **VKRSS_ACCESS_TOKEN**
 - **VKRSS_DOMAIN** 
 - **VKRSS_FILTER**
 - **VKRSS_COUNT**
 - **VKRSS_PROXY**
 - **VKRSS_NEED_IMAGES**
 - **VKRSS_NEED_AUDIOS**
 - **VKRSS_NEED_VIDEOS**
 - **VKRSS_NEED_DOCS**

Docker exposed port - 8888

Example RSS-feed here: https://riftbit.com/vk_rss
